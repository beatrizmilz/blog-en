Boas vindas!

É muito importante proporcionar uma experiência livre de assédio para todas as pessoas participantes. Por favor, leia o Código de Conduta da R-Ladies.

Pré-requisitos

Para praticar o que faremos na apresentação, esses são os pré-requisitos:

  • Instale o R
  • Instale o RStudio
  • Instale o Git, neste link escolha o sistema operacional que você utiliza, e instale utilizando o arquivo baixado.
  • Crie uma conta no GitHub, é gratuito.
  • Instale alguns pacotes:
install.packages("usethis")
install.packages("rmarkdown")
